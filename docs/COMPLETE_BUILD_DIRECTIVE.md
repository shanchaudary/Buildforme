# Buildforme — Corrected Complete-Build Directive

**Status:** canonical product law for stages 6–10  
**Product:** Generic Founder Control Plane for supervising AI software engineering  

## Product identity

Buildforme supervises AI engineering across **multiple projects, repositories, stacks, and companies**.

It must **not** be designed specifically for Cascada, one SaaS product, one language, one framework, one repository, one provider, one OS, or one company.

Cascada is the first proving project **after** Buildforme reaches required execution capability. Cascada validates Buildforme; Cascada must not define Buildforme’s architecture.

Generic core concepts only: projects, repositories, roadmaps, project truth, tasks, packets, providers, transports, execution runs, worktrees, evidence, verification, reviews, approvals, founder decisions, audit records, briefings.

Repository-specific behavior is supplied through project configuration, repository inspection, policy profiles, task contracts, and adapter interfaces—not hardcoded product logic.

## Delivery model

Controlled numbered stages. Each stage:

1. One complete architectural objective  
2. End-to-end implementation  
3. Backend, storage, API, CLI, UI, tests, governance, audit, docs where applicable  
4. Independently reviewed  
5. Adversarial testing  
6. Accepted before the next stage  
7. No intentionally unwired required capability  
8. No fake/placeholder/mock path inside declared stage scope  

The founder authorizes the **stage**. Engineering manages bounded implementation work.

## No postponement rule

Do not defer required stage capabilities as “future work / post-MVP / optional” when the current stage depends on them or implies they exist.

## Cascada’s role

Proving project only. Cascada details live in Cascada project config / truth / roadmap / verification profile / packets—not Buildforme core.

## Whole-system completion

The assignment does not end at Stage 6. Continue through Stage 10 until the full control-plane loop is real, portable, and non-stubbed.
